**Welcome to StudyMon**

**About**

StudyMon is a React/Vite productivity app that stores your data locally in the browser.

This project contains everything you need to run the app locally.

**Edit the code in your local development environment**

Any change pushed to the repo will update the application behavior locally.

**Prerequisites:**

1. Clone the repository using the project's Git URL
2. Navigate to the project directory
3. Install dependencies: `npm install`

Run the app: `npm run dev`

For project-specific support, check local repo documentation or ask your team.
